## 📝 Descripción del Proyecto
[cite_start]El proyecto consiste en el diseño y desarrollo de un **Sistema Web de Control de Planilla para la Fundación Bandera Blanca**[cite: 1]. [cite_start]El objetivo principal es automatizar los procesos de gestión de nómina, incluyendo el cálculo de salarios, liquidaciones y aguinaldos, para reducir errores humanos y mejorar la eficiencia administrativa de la organización[cite: 54, 690].
