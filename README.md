# SWCP-FBB: Sistema Web de Control de Planilla

Este repositorio contiene el prototipo funcional desarrollado como **Trabajo Final de Graduación** para optar por el grado de **Bachillerato en Ingeniería Informática** en la **Universidad Internacional de las Américas (UIA)**.

## 📝 Descripción del Proyecto
El proyecto consiste en el diseño y desarrollo de un **Sistema Web de Control de Planilla para la Fundación Bandera Blanca**. El objetivo principal es automatizar los procesos de gestión de nómina, incluyendo el cálculo de salarios, liquidaciones y aguinaldos, para reducir errores humanos y mejorar la eficiencia administrativa de la organización.

## 🛠️ Tecnologías Utilizadas
* **Lenguaje:** Python
* **Framework Backend:** Flask
* **Base de Datos:** MySQL
* **Frontend:** HTML5, CSS3, JavaScript, Bootstrap y Jinja2
* **Metodología:** Ágil (Scrum)
